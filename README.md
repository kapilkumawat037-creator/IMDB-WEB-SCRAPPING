This project is a Python-based web scraper that extracts movie information from IMDb and converts it into structured datasets for analysis.

The scraper collects details such as:
- Movie Title
- Rating
- Genre
- Release Year
- Cast
- Duration
- Reviews
- Votes

The data can be exported to CSV/Excel for further analysis, visualization, or machine learning tasks.

Tech Stack:
- Python
- Requests
- BeautifulSoup
- Pandas

This project demonstrates practical skills in web scraping, data cleaning, and data preprocessing.
